# Day 5/30 - Cat Photo App | freeCodeCamp Responsive Web Design
#30DaysOfCode #HTML #WebDev


## 📖 Project Overview
Built a complete Cat Photo App webpage using only HTML5. The app demonstrates semantic structure, accessibility basics and form handling through 42 progressive steps. 


## 🎯 What I Learned - Day 5
Core HTML5 concepts practiced in this build:

**Structure & Semantics**
- Document setup: `<!DOCTYPE html>`, `<html lang="en">`, `<head>`, `<body>`
- Semantic tags: `<main>`, `<section>`, `<footer>`, `<figure>`, `<figcaption>`
- Headings hierarchy: `<h1>` to `<h3>` for proper document outline

**Links, Images & Text**
- Anchor tags: internal links, external links, turning images/text into links
- Images: `src` + `alt` attributes for accessibility
- Text formatting: `<p>`, `<em>`, `<strong>`, lists `<ul>`, `<ol>`, `<li>`

**Forms & Accessibility**
- Form elements: `<form>`, `<fieldset>`, `<legend>`, `<input>`, `<label>`, `<button>`
- Input types: `radio`, `checkbox`, `text` + attributes `name`, `value`, `checked`, `required`
- Meta tags: `<meta charset="utf-8">` for character encoding
- Accessibility: proper `alt` text + label association

**Debugging Skills**
- Fixing syntax errors: missing quotes, unclosed tags, incorrect nesting
- Reading FCC error messages to identify exact line issues

## ✨ Features
- Responsive image display with captions using `<figure>` + `<figcaption>`
- Lists for "Things cats love/hate" 
- Accessible survey form for cat personality traits
- Semantic HTML structure throughout
- Meta charset for proper text rendering
